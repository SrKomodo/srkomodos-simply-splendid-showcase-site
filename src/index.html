Atrocities that I made

Boolean Formula Solver (link) (code)

Some time ago i was learning about boolean formulas and making really boring tables and diagrams about them. There was a website that did the formula for you but it displayed it in horrible ascii art, so I decided to make one myself!

This website let's you input a formula, and it will calculate the table and render the circuit for it. You can also input a table and the program will generate a formula that matches the table's results.

Mouse Pet (link) (code)

This is a simple Firefox webextension that adds a cool little snake following your cursor. You can change the length of the snake, and choose from a list of skins to use. Also you can make the snake disappear when the mouse isn't moving so that it doesn't bother you.

Lucas's Rulers (link) (code)

On January of 2018 Vsauce uploaded a video about Genaille-Lucas rulers, a set of rulers that let you multiply numbers without actually doing the math.

You see, the problem with this video is that it was uploaded on the DONG channel (which stands for Do Online Now Guys) and those rulers aren't really a DONG. So I made it into a DONG!

The website let's you input two numbers, and the website will show you an animation of the rulers doing the calculation. It also works on mobile and let's you download it as an app. But the best thing about the website is that the Sauce himself liked it.

Decrypter (link) (code)

I really like ARGs and mystery solving (if you like that kind of stuff check out Gravity Falls it's an amazing show) so I decided to make a website to easily decrypt text with various common ciphers.

Epicycler (link) (code)

An epicycle is the movement of a point on a rotating circles. If you add rotating circles to those rotating circles you can get some pretty interesting patterns, so I made a website that randomly generates epicycles and let's you see how they spin around.

Most procastination blockers are simply browser extensions that block certain websites at certain times. The problem with this is that i can just open up the website on another browser and evade the blocker, also it would be cool if we could also block certain programs from running.

YAPB (Yet Another Procastination Blocker) fixes both of these problems, by blocking websites globally by editing the hostfile on your computer. Also it stops programs from running.
